# Cyber Security Incident Response and Digital Forensics Investigation

## Project Overview

This project presents a simulated Cyber Security Incident Response and Digital Forensics Investigation for a small IT organization. It demonstrates the complete incident response lifecycle, including incident identification, evidence collection, forensic analysis, containment, recovery, and security recommendations.

## Project Objectives

- Understand the Incident Response lifecycle.
- Investigate a simulated ransomware attack.
- Collect and preserve digital evidence.
- Perform root cause analysis.
- Recommend security improvements.

## Incident Scenario

ABC Technologies Pvt. Ltd. experienced a simulated ransomware attack through a phishing email. The infected system was isolated, digital evidence was collected, and a forensic investigation was conducted to identify the root cause and recover the affected systems.
